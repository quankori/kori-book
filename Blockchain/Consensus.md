# Blockchain Consensus

## Proof of Work (computer have more resource to win).

- Use power of computer to execute algorithm. (computer have more resource to win).

- Miner will be find nonce correct such that the first character of the hash string generated with that nonce must be less than target (hexadecimal).

- The target is calculated by each node in the network independently (follows certain rules).

- Verify nonce will fast find nonce.

- The smaller the target (hexadecimal) will take more time to mine. (target 0x0001 will take more time than 0x010000).

## Proof of Stake (pay to win)

- Algorithm use random and compare with money use have to execute.

- Algorithm use two character in hash string and multiply with target, after calculated it will compare with amount of money user have. If multiplication result less than money user have, user winner node.
